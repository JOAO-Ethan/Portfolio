Zelda-like video game

JavaFX Java Personal Project

This is an ongoing personal project which consists of the creation of a JavaFX video game inspired by the first The Legend of Zelda game.

It will be made of a screen from the original game in which random enemies (they also come from the original game) will spawn by waves.
therefore, the goal will be to gain points by killing those enemies and do so without dying.

Some items would appear when you defeat an enemy, such as hearts, in order to heal oneself, or stopwatch, in order to stop time.

Below this paragraph, you can find pictures from the original game that I use to create my own game : an area I’ll later recreate in JavaFX and a sprite sheet showing different enemies.
