PHP Laravel School Project Git

This is a school project made in November 2022 during my three-year university degree in IT. It is a group project in collaboration with some students from another three-year university degree.
This project consists of a numeric art gallery website, which contained rooms and differents works per room.

It was also possible, when an user was logged in, to “like” a work, leave a comment and even publish your own work in a special room.

The back-end part of this project was carried out on Laravel by both my coworkers, Florentin Wujciak and Tom Decary, and I. The front-end part and the website’s topic was made by the other students whereas the web integration was realized by everyone.
