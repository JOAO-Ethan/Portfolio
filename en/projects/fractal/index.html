Fractals generator library

Java School Project Git

This is a school project made in October 2022 during my three-year university degree in IT. It consists of the creation of a Java library that generate fractals.

This project was created in group with 2 friends of mine : Florentin Wujciak and Tom Decary.
Thanks to this, we worked on design patterns and complex numbers, given that we generated the Mandelbrot and Julia. We also created the Sierpiński carpet.
