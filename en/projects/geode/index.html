Geode Project

SQL Java JavaFX Database Personal Project

This is an ongoing personal project which consists in the creation of a database in order to organize gemstones collections.

With this project, you will be able, thanks to a JavaFX application, to add a specimen in the database by giving a picture, it mineral species, when it was acquired and where.
This application will also allow you to delete a specimen and to list all specimens in the database. Another possibility will be to sort specimens by species, its origin or acquisition’s date.

Mineral species in the database are based on those indexed by the International Mineralogical Association.

At last, another feature will be to identify a mineral by giving some parameters, like its scale on the Mohs scale of mineral hardness.
