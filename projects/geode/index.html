SQL Java JavaFX Base de données Projet Personnel

Ceci est un projet personnel en cours consistant en la réalisation d’une base de donnée pour l’organisation de collections minéralogiques.

Il permettra, à termes, d’ajouter des spécimens dans la base de données à l’aide d’un logiciel dédié développé à l’aide de JavaFX en précisant les différentes espèces minérales la constituant ainsi que son lieu de provenance et sa date d’acquisition. Il sera aussi possible de joindre une image du spécimen en question.
La-dite application permettra de lister les spécimens en collection, d’en ajouter ou d’en supprimer. Il sera aussi possible de trier les spécimens en fonction d’une ou plusieurs espèces minérales, par date d’acquisition ou lieu de provenance.

Les espèces minérales indiquées dans la base de donnée se baseront sur celles répertoriées par l’IMA.

Une fonctionnalité permettant d’identifier une espèce minérale en fournissant divers paramètres est également prévue.
